# Web Parser

Small MCP helper for fetching, summarizing and inspecting web pages for AI clients.

## What it does

- fetches raw HTML
- summarize websites, forums, news, etc.
- extracts cleaned page text
- extracts links
- helps inspect suspicious websites without running JavaScript

## Run

1. Install dependencies with uv
2. Start the server from this folder

This is part of the larger mcp-threathunting project.
